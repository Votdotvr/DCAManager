# DCA Manager

![Banner](https://raw.githubusercontent.com/Votdotvr/Photos/640c24dd8ec638f6658b754bf655e6be01ed64c0/DCAManager.png?token=CE5KCE3H3VOQP3S5JLA5IS3KLVOFC)

DCA Manager is a free, open‑source mod management tool for any game.  
It lets you add, remove, enable, disable, and organize mods with a simple UI.

## ✨ Features

- Add mods with custom names
- Disable mods (moves them to `disabledmods/`)
- Enable mods (moves them back into the game folder)
- Remove mods cleanly
- JSON‑based mod storage
- Simple, dark‑mode UI
- Built with Python + Tkinter

## 📁 Folder Structure

```
DCAManager/
│  DCAManager.py
│  mods.json
│
├─ disabledmods/
    (mods go here when disabled)
```

---

## 🚀 Installation

1. Download the latest release from GitHub.
2. Extract the ZIP.
3. Run `DCAManager.py`.
4. Add your mods and start managing them.

---

## 🛠 Requirements

- Python 3.10+
- Tkinter (included with Python)
- Windows (recommended)
